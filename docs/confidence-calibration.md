# Confidence Calibration

Confidence is evidence-derived, never a probability claim. JAN, exact product name, alias, maker, type, volume, ABV, repeated multi-photo text, confirmed visual reference, history, and fuzzy retrieval are tracked separately.

Single weak-source candidates are capped; mismatches subtract points; high confidence requires strong identity or multiple independent evidence kinds. The final path abstains on low score or inadequate candidate margin. Validation reduced false positives from 54.6% in cycle 1 to 9.1% in cycle 6, while unknown-candidate rate fell from 63.2% to 0%. Sealed holdout false-positive rate was 0% with 81.8% abstention.

The tradeoff is deliberate: low recall is visible to the user as “特定できませんでした”, not concealed by unrelated candidates.
