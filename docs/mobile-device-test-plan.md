# Mobile device test plan

Use Settings > device validation mode on a physical device. Record pass, fail, or not run for photo library, HEIC, ten-file selection, first OCR model load, cached offline OCR, keyboard, safe area, PWA installation/restart, forced termination draft recovery, service-worker update, backup and restore.

Do not describe WebKit/Chromium emulation as a physical-device result. Exported JSON contains browser diagnostics and check results, but never image bytes or the Rakuten application ID.
